# 基礎演算

演算と対応する演算子は以下の通りです。

| 演算子 | 意味   |
| ------ | ------ |
| +      | たす   |
| -      | ひく   |
| \*     | かける |
| /      | わる   |
| %      | 余り   |
| ^      | 累乗   |

```lua
1 + 2 * 3 / 4
```

👉`2.5`
算数と同じ計算順序ルールが適用されるので注意 😮

```lua
10 % 3
```

👉 `1`

```lua
10 ^ 3
```

👉 `1000`

# 切り上げ/切り捨て

標準関数を使います。
小数点以下が切り上げ/切り捨てされます。

| 関数  | 意味                                        |
| ----- | ------------------------------------------- |
| ceil  | 切り上げ、天井(ceil)に引き上げるイメージ 💭 |
| floor | 切り捨て、床(floor)に引き下げるイメージ 💭  |

```
ceil(1.5)
```

👉 `2`

```
floor(1.5)
```

👉 `1`

## 特定の桁数で切り上げ/切り捨て

小数点第 XX 位で切り上げ/切り捨てを行いたい場合は、
その桁数分だけ数値を 10 の累乗でずらしてから
切り上げ/切り捨てを行い、その後元の桁数に戻します。

**1.2345を小数点第2位で切り上げ**
```lua
ceil(1.2345 * 10^2) / 10^2
```
👉 `1.24`

**1.2345を小数点第3位で切り捨て**
```lua
floor(1.2345 * 10^3) / 10^3
```
👉 `1.234`

## 四捨五入

四捨五入を行うには`0.5`を足してから切り捨てます。

**1.5を四捨五入**
```lua
floor(1.5 + 0.5)
```
👉 `2`

**1.4を四捨五入**
```lua
floor(1.4 + 0.5)
```
👉 `1`

# 条件分岐

条件によって計算を分けるには`iif`関数を使用します。
:::details iif とは
Immediately IF の略だと思います 🫡
:::
`iif(条件, 真の場合, 偽の場合)`の形で記述します。
Excel の IF 関数と同じです

条件を記述するための関係演算子は以下の通りです。

| 演算子 | 意味                       |
| ------ | -------------------------- |
| <      | 左辺が小さい               |
| >      | 左辺が大きい               |
| <=     | 左辺が小さい、または等しい |
| >=     | 左辺が大きい、または等しい |
| ==     | 等しい                     |
| ~=     | 等しくない                 |

```
iif (1 > 1, 1, 0)
```
👉 `0`

```
iif (1 >= 1, 1, 0)
```
👉 `1`

```
iif (1 == 2, 1, 0)
```
👉 `0`

```
iif (1 ~= 2, 1, 0)
```
👉 `1`

:::message
実際には`1 < 2`の用に数値同士を条件に設定することはなく
`Transform1.Size < 0.5`のようにコントロールの値と比較します 😤
:::

## 3 分岐以上の分岐

3 分岐以上させたい場合`iif`をネストさせます。
ただし複雑になってわかりにくいので[マルチステートメント](#マルチステートメント)を使用したほうが良いです 😑

Transform1 の size が`0.5未満`, `0.5-1.0`, `1.0より大きい`のどれに当たるかを
判定し、それぞれ 0, 1, 2 という値に変換する例

```
iif (Transform1.Size < 0.5, 0, iif(Transform1.Size <= 1.0, 1, 2))
```

**右**

**左**


**上**
```lua

```

**下**
```lua
Text1.Output.DataWindow.bottom / Text1.Output.Height
```

Scriptから描画範囲を取得したい場合

```lua
print("右: " .. (Text1.Output[0].DataWindow[3] / Text1.Output[0].Width))
```

## 幅/高さ

右から左を引くと幅

`Text1`ノードのテキストエリアの幅/高さを取得する例

```lua

```

**高さ**
```lua

```

# 🐦 応用

## クリップの長さ

クリップ末尾フレーム数からクリップ先頭のフレーム数を引くと
クリップの長さになります。

```lua
comp.RenderEnd - comp.RenderStart
```

## フレーム数から秒に変換する

フレーム番号をフレームレートで割ればフレーム時間(秒)になります

現在のフレーム位置の時間を取得する例

```lua
time / comp:GetPrefs("Comp.FrameFormat.Rate")
```

先頭フレームが 0 では無いことも考慮する場合は以下の計算式になります。

```lua
(time - comp.RenderStart) / comp:GetPrefs("Comp.FrameFormat.Rate")
```

## 特定の再生位置で処理を分岐させる

特定の再生位置で処理を分岐させたい場合
[条件分岐](#条件分岐)と[時間計算](#フレーム数から秒に変換する)を組み合わせます。
また、複雑になるので[マルチステートメント](#マルチステートメント)の出番です ❗️

クリップ 2 秒までは`0`, 2 秒後は`1`に設定する例

```
:
frameRate = comp:GetPrefs("Comp.FrameFormat.Rate")
targetTime = 2.0
firstHalfValue = 0
secondHalfValue = 1

currentTime = (time - comp.RenderStart) / frameRate

return iif(currentTime < targetTime, firstHalfValue, secondHalfValue)
```

<br/>

クリップ 40%までは`0`, 40%以上は`1`に設定する例

```
:
targetRatio = 0.4
firstHalfValue = 0
secondHalfValue = 1

currentRatio = ((time - comp.RenderStart) / (comp.RenderEnd - comp.RenderStart))

return iif(currentRatio < targetRatio, firstHalfValue, secondHalfValue)
```

## Shape, Mask 座標変換

`Rectangle`や`Ellipse`、`Polygon`などの**Mask系ノード**と、`sRectangle`や`sEllipse`、`sPolygon`などの**Shape系ノード**では、座標系の定義が異なります。

Mask系ノードは、左下を原点 `(0.0, 0.0)`、右上を `(1.0, 1.0)` とする座標系です。
このとき、アスペクト比にかかわらず右端・上端ともに `1.0` となる点に注意してください☝️
つまり、正方形以外の解像度では縦横で値の「重み」が異なります。そのため、幅と高さに同じ値を設定しても、横長のアスペクト比なら横長の四角形になります。

対してShape系ノードは、中央を原点 `(0.0, 0.0)` とし、右端を `0.5`（左端を `-0.5`）とする座標系です。
こちらは左右方向を基準として、縦方向も同じ重みで扱われます。そのため、幅と高さに同じ値を設定すれば、アスペクト比にかかわらず正方形になります✨

これらの違いがあるため、Mask系とShape系の間で値をやり取りしたいときは、座標変換を行う必要があります。

![Mask vs Shape](/images/articles/expression/mask-vs-shape.png)
_Mask vs Shape_


### Mask系 -> Shape系への変換

ポイントは高さ方向に`アスペクト比の逆数`を掛け合わせることです👍
位置の場合さらに`0.5`を引くことで変換します

以下は変換元を`Rectangle1`とした例

**Shape X**
```lua
Rectangle1.Center.X - 0.5
```

**Shape Y**
```lua
(Rectangle1.Center.Y - 0.5) * (Rectangle1.MaskHeight / Rectangle1.MaskWidth)
```

**Shape Width**
```lua
Rectangle1.Width
```

**Shape Height**
```lua
Rectangle1.Height * (Rectangle1.MaskHeight / Rectangle1.MaskWidth)
```

![Mask -> Shape](/images/articles/expression/convert-mask-to-shape.png)
_Mask -> Shape_

### Shape系 -> Mask系への変換

ポイントは高さ方向に`アスペクト比`を掛け合わせることです👍
位置の場合さらに`0.5`を足すことで変換します

以下は変換元を`sRectangle1`, `sRender1`とした例

**Mask X**
```lua
sRectangle1.Translate.X + 0.5
```

**Mask Y**
```lua
(sRectangle1.Translate.Y * (sRender1.Output.Width / sRender1.Output.Height)) + 0.5
```

**Mask Width**
```lua
sRectangle1.Width
```

**Mask Height**
```lua
sRectangle1.Height * (sRender1.Output.Width / sRender1.Output.Height)
```

![Shape -> Mask](/images/articles/expression/convert-shape-to-mask.png)
_Shape -> Mask_

## Python script

Expressionでは使えないのですが(使い方がわからない😭)
コントロールの`On Change`やボタンコントロールの`Execute`では
Pythonを使うことも出来ます✨✨

スクリプトの先頭を特定の文字列から始めると、その言語で記述することが出来ます。

| 先頭文字列 | 言語 |
| ---- | ---- | 
| !Lua: 　| Lua |
| !Py2:   | Python 2 |
| !Py3:   | Python 3 |

:::message alert
上記の表では表現できていませんが `:` の後に半角の空白が必要です！！
:::


### サンプル

```python
!Py3: 
import http.client
import json

connection = http.client.HTTPSConnection("pokeapi.co")
connection.request("GET", "/api/v2/ability/209") # 209:Mimikyu
response = connection.getresponse()
bytes = response.read()
response_json = bytes.decode("utf-8")
json_dict = json.loads(response_json)
flavor_text = json_dict["flavor_text_entries"][0]["flavor_text"]
comp.Text1.SetInput("StyledText", flavor_text)
```
