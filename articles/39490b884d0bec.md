# 🐥 はじめに

みなさん、こんにちは。[Mug](https://www.youtube.com/channel/UCuhx0M-PBn4qJ-SUKQ6gVaA)です🐼

DaVinci Resolveには[We Suck Less](https://www.steakunderwater.com/wesuckless/)というコミュニティーサイトがあります
これは公式なものではありませんが、Fusion関係の情報があつまる
一番大きなコミュニティーの一つです
この記事では`We Suck Less`への新規ユーザー登録手順を解説します😤

`We Suck Less`ではFusionのマクロファイルや
その他お役立ちツールがスレッドに添付されていたりしますが、
それらをダウンロードするにはログインが必要になってきます

また、別途記事を作成予定の`Reactor`のインストールや、`Reactor`へのエフェクト登録申請でも
ログインが必須となるので登録しておきましょう👍

:::message
この記事では、2026/4/13現在の手順で解説しています
:::

# 1. WeSuckLessへの新規ユーザー登録

[WeSuckLess](https://www.steakunderwater.com/wesuckless/)にアクセスして右上の
✅チェックボックスのようなアイコンをクリックし、アカウント登録を行います


![Register](/images/articles/register-wsl-user/register-wsl.png)
_Register_

## 2. 利用規約確認

利用規約が色々書いてあるので**よく読み**、同意👍して先に進みます

:::message alert
当たり前ですがちゃんと読みましょう
特にAIによる学習目的や商業目的でのコンテンツの収集・利用は禁止されているので注意してください
:::

![Agree terms](/images/articles/register-wsl-user/agree-terms.png)
_Agree terms_

## 3 ユーザー情報入力

必要事項を入力します、必須なのはユーザー名、メールアドレス、パスワードくらいです
あとはお好みでどうぞ🍌

![Input user information](/images/articles/register-wsl-user/registration1.png)
_Input user information_


## 4. 超難関クイズ

最後にクイズが出題されています
表示されているものがFusionノードなのかどうかを振り分けます
ドラッグ＆ドロップで移動させます

これは🤖ロボットによる自動登録回避ですが
Fusionを触っていないと人間でも難しいかも🤣

クイズが解けたら`Submit`ボタンを押します

![Answer the quiz](/images/articles/register-wsl-user/registration2.png)
_Answer the quiz_

![Account created](/images/articles/register-wsl-user/registration-complete.png)
_Account created_

## 5. アクティベーション

入力したメールアドレスにメールが届くので、メール内のリンクをクリックして
🔥アクティベーションします

:::message
メールが届かない場合は、迷惑メールフォルダを確認してください👀
:::

![Activation](/images/articles/register-wsl-user/wsl-activation.png)
_Activation_

## 6. ログイン確認

右上の電源ボタンをクリックして、登録したユーザー名、パスワードでログインできることを確認します
右上のアイコンが変化して、色々出ていればログイン成功です🎉
これでアカウント登録完了です✨
